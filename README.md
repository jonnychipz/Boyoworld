# BOYOWORLD

A five-level promotional browser game for BOYO, built as a static Phaser 3 experience for GitHub Pages.

The repository vendors Phaser locally and includes an authorized visual library sourced from BOYO's official YouTube, TikTok, Linktree, and storefront. Each level uses layered music-video frames rather than a generic game backdrop.

## Play locally

Open `index.html` directly, or run:

```powershell
npx serve .
```

## Controls

- Move: arrow keys or WASD
- Action/jump/fire: Space or the on-screen action button
- Pause: P or Escape
- Restart: R
- Rhythm lanes: D, F, J, K
- Memory pads: arrow keys or click/tap

Sound is synthesized locally with the Web Audio API. Music-video rewards use privacy-enhanced official YouTube embeds and require an internet connection.

## Five worlds

1. **Red Brick Run** — cinematic street runner
2. **Spitfire Beat** — four-lane rhythm game
3. **Fuming Escape** — top-down urban chase
4. **Raw Pap Arena** — 360-degree survival shooter
5. **Pay Me Panic** — visual memory reaction game

The playable BOYO is a layered Phaser rig: the supplied mask photography is combined with an articulated scarf, torso, arms, legs, gait, jump, dance, recoil, shadow, and speech system.
